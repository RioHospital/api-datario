API - RioHospital - DataRio

A API fornece métodos para acessar dados dos hospitais e outros estabelecimentos de saúde do Rio de Janeiro.

Dependências: API data.rio ("http://dadosabertos.rio.rj.gov.br/apiSaude/apresentacao/rest/index.cfm/estabelecimentos")

Métodos:

HTTP GET: /hospitals
Descrição: Retorna uma lista com os hospitais cadastrados a serem exibidos de forma filtrada pela localização/raio no app, no formato:
{"name": NOME, "address": ENDEREÇO, "neighborhood": BAIRRO, "postalCode": CEP, "phone": TELEFONE, "latitude": LATITUDE, "longitude": LONGITUDE}
